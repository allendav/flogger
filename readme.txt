== Description ==

Tag your posts with the exercises you did that day, including how much (e.g. steps, minutes, reps, etc.).  Exercises are displayed at the bottom of the post.  Designed to encourage daily blogging and exercise.

== Installation ==

1. Upload/install the plugin
2. Activate the plugin
3. Add some exercise tags under Posts > Exercises
4. Write a blog post
5. Exercise!
6. Tag your post with the exercises you did that day.  Add how much if you'd like too.

== Frequently Asked Questions ==

= Will this actually get me to blog and exercise more? =

Yes!  At least it does for me!  Your mileage may vary.
